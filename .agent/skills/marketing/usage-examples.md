## Skill Usage Checklist

When using the AdSync marketing skill, always:

- [ ] Identify the primary pillar (Payment / Creative / Simplicity)
- [ ] Check relevant reference files before writing
- [ ] Apply brand voice guidelines (see brand-voice.md)
- [ ] Include specific details (Naira amounts, timeframes, Nigerian context)
- [ ] Add clear CTA (one action, not multiple)
- [ ] Verify against anti-scam filter (no get-rich-quick promises)
- [ ] Review for local context (WhatsApp, mobile-first, Paystack)
- [ ] Ensure tone matches audience stage (new vs. churned vs. active)
- [ ] Test that competitive positioning is honest and specific
- [ ] Check that copy is scannable (short paragraphs, bullet points)
